== Description ==

CedCommerce Jet-Integration fro WooCommerce extension interacts with Jet Marketplace to integrate the synchronized product listing between WooCommerce and Jet.com retailers. After extension installation, merchant can create Jet Categories & their dependent attributes on WooCommerce store. The process will enable merchant to configure the desired product category into WooCommerce, for automatic submission of the selected product to the same Category on Jet.com.

[youtube https://www.youtube.com/watch?v=_5VsmjynifA]

== Screenshots ==

1. “jet details” that allows merchant to fill the basic jet configuration details which merchant gets from jet.com.
2. “Return Location settings” the merchant needs to enter the address where product has to be returned if any return is generated for orders.
3. “Extra settings” which provide settings for “automatic order acknowledge” during order creation in woocommerce , another settings is “synchronize jet product” with  woocommerce product update , ”Archive child product” also on archive of parent and “Update Inventory” with default quantity when product quantity become zero.
4. To upload a product on jet.com, a jet category must be assigned to it. For this the merchant needs to map your WooCommerce category with jet category. You can map your WooCommerce category with jet under “Category Mapping” section.
5. This tab will enable you to create profile and where you can set the common attributes value for product and then assign profile to product from “Manage Product” Section. While creating profile you can also map your attributes field with any other meta field. 
6. Using profile you can handle common attributes values easily and also remove the redundancy of data.
7. Under “jet Attributes” section all attributes value is compulsory to fill and also select anyone mapped category according to you product requirement ,if you skip to enter any value then you can’t upload your product on jet.com.
8. you can also send additional value too for product . you can see under “jet Extra attributes” section .
9. you can set the shipping exception for multiple fulfillment. To set the shipping exception for product , click on “Shipping Exception” tab.
10. “Return Exception” tab ,before set return exception ,be confirm that you have fill all the details for “return location settings” such as return location id, first address, city, state and pin code under “jet configuration” tab . 
11. You can create Simple product and assign the regular price to that product.
12. You can close the product by selecting 'Archive' option from the dropdown.
13. You can open the closed product by selecting the 'Unarchive' option from the dropdown.
